## Installation

#### 1. Clone the Repository
```
git clone https://github.com/ZAKHAROV-Artem/tamagui-tutorial .
cd tamagui-tutorial
```
#### 2. Install Dependencies
```
npm i
```
#### 3. Prebuild the Project
```
npm run prebuild
```
#### 4. Run on Android or Ios
```
npm run andriod 
```
```
npm run ios
```
